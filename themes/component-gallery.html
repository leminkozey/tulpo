Iron Rules

These rules apply to EVERY component. No exceptions.

RuleDetails
NO GLOW No colored box-shadows on hover/focus/active. Only exception: cards have a neutral shadow (rgba(0,0,0,0.3)).
Focus = Border only Input focus only changes border-color to --text-muted. No ring, no outline, no shadow.
Subtle Hover Buttons: translateY(-1px). Cards: translateY(-2px). Messages: bg change only. Nothing flashy.
Active = Press Buttons: translateY(1px) on :active. Provides haptic feedback without glow.
Transitions 0.1s - 0.2s ease. Never longer. Never bouncy (except toggle thumb cubic-bezier).
Spacing 4px grid basis. Gaps: 4px (tight), 8px (compact), 12px (default), 16px (spacious).
No custom colors Only use defined CSS custom properties. Never hardcode hex values in components.
