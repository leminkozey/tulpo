# Tulpo Roadmap

## Known Bugs

- [ ] @mention tag background not vertically centered around text (padding/alignment issue in `renderContent`)
- [ ] Emoji reactions not perfectly vertically centered in pills (later fix)

## Chat Features

- [x] Message editing (own messages)
- [x] Message deletion (for everyone / for me)
- [x] Reply to messages (quote + reference)
- [x] Image sending (paste, drag & drop, file picker)
- [x] Emoji picker & sending
- [x] GIF gallery (GIPHY integration)
- [x] File uploads (documents, audio, etc.)
- [x] Reactions (emoji reactions on messages)
- [x] Typing indicators
- [x] Per-chat message drafts
- [x] Jumbo emoji display (1-5 emoji-only messages)
- [ ] Search messages

## Social

- [x] Friends system (add, accept, decline)
- [x] Block / unblock users
- [x] Mute / unmute users
- [x] Report users / messages / files
- [x] Real-time presence (online/offline)

## Infrastructure

- [ ] Servers (create, join, invite)
- [ ] Channels (text, voice) within servers
- [ ] Roles & permissions
- [x] User settings page
- [ ] User profiles (avatar upload, bio, status)
- [ ] Themes (Frost theme)

## Customization & Theming (most requested — 11k+ votes on Discord feedback)

- [ ] Full theme engine — colors, backgrounds, fonts, layout density via CSS Custom Properties
- [ ] Server-specific themes — admins set look & feel per server (colors, backgrounds, branding)
- [ ] Theme store — community-created themes: browse, share, rate, download
- [ ] Profile customization — banners, accent colors, custom borders (no paywall)
- [ ] Per-server profiles — unique avatar, display name, bio per server
- [ ] UI density settings — compact, default, spacious
- [ ] Custom CSS support — power users can inject their own styles
- [ ] Font selection — let users pick their UI font

## Voice & Audio

- [ ] Whisperlists — hotkey to talk to specific people/groups, not the whole channel
- [ ] Per-user volume control — individual volume sliders per member
- [ ] Call time counter — show how long you've been in a voice chat
- [ ] Low-latency audio — prioritize voice quality over features (TeamSpeak-level)

## Privacy & Data Sovereignty

- [ ] No ID verification — self-hosted = no government ID needed
- [ ] E2E encryption — optional per channel/DM (already decided)
- [ ] Full data ownership — admin controls what's stored, exported, deleted
- [ ] No telemetry — zero data sent to third parties

## Search & Navigation

- [ ] Cross-channel search — search across all channels at once
- [ ] @mentions overview — see all your mentions in one place ("For You" view)
- [ ] Advanced search filters — by user, date range, has:file, has:image, etc.

## Moderation & Admin

- [ ] Granular permissions — fine-grained role system beyond Discord's model
- [ ] Permissions viewer — see who has what rights at a glance
- [ ] AutoMod — configurable spam/content filters
- [ ] Audit log — who did what, when

## Channel Features

- [ ] Threaded conversations — sub-discussions without leaving the channel
- [ ] Whisper/scoped messages — talk to a subset of channel members
- [ ] Better pinned messages — separate from thread pins, with previews
