# Discord Alternative — Projektübersicht

## Vision
Eine self-hosted Discord-Alternative die **stupide einfach zu hosten** UND **schön** ist. Das hat keiner der [[02 - Bestehende Alternativen — Analyse|bestehenden Alternativen]].

## Warum jetzt?
Siehe [[04 - Marktanalyse & Timing]] für die vollständige Analyse.

- Discord führt **Facial ID Verification** global ein (ab ~Mitte 2026)
- "Discord alternatives" Suchen um **10.000%** gestiegen
- TeamSpeak meldet "incredible surge" neuer User
- 16.000+ Upvotes auf r/pcmasterrace für "Zeit für eine Discord Alternative"
- 5.000+ Upvotes auf r/selfhosted für Discord-Replacement Thread
- Die Community ist **aktiv am Suchen** — der Markt ist jetzt offen

Mehr Details: [[01 - Warum Leute Discord verlassen]]

## Kernproblem
Alle [[02 - Bestehende Alternativen — Analyse|bestehenden Alternativen]] scheitern an mindestens einem dieser Punkte:
1. Zu kompliziert zum Hosten (Matrix)
2. Schlechte UX / hässliche UI (Matrix, TeamSpeak, Mumble)
3. Fehlende Core-Features (Voice, Screensharing)
4. Kein gescheiter Mobile-Support
5. Paywall für Basic-Features (Mattermost, Rocket.Chat)
6. Nicht user-friendly genug für Normies ("Wife Test")

## Differentiator
- **Ein Docker-Command zum Starten** — kein PhD nötig
- **Moderne, cleane UI** — nicht wie 2004
- **Open Source + Self-Hosted** — Daten bleiben beim User
- **Kein Facial ID, kein Bullshit**
- **Durchsuchbare Inhalte** — Wissen nicht in Walled Gardens eingesperrt
- **Forum/Knowledge-Base Features** — was [[06 - Zitate & Stimmung|7527 Leute wollen]]
